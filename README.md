# AgendamientoCitas.Frontend

Frontend React para consumir la Minimal API de agendamiento de citas.

## Ejecutar

```powershell
npm install
npm run dev -- --host 127.0.0.1
```

La API debe estar disponible en `http://localhost:5026`. Si cambia el puerto, crea un archivo `.env` con:

```text
VITE_API_URL=http://localhost:5026
```
